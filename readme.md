#weazm laravel boilerplate


##快速安装
```
wget https://coding.net/u/troylee/p/weazm-laravel-boilerplate/git/archive/master

unzip troylee-weazm-laravel-boilerplate-master {your project name}

cd {your project name}

git init (初始化git，添加你项目的repo url)

```

- `composer install`
- `composer dump-autoload`
- `chmod -R 777 app/storage`
- `cd public`
- `bower install`
